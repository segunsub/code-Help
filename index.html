Search Bar
The search bar can be used to search up code questions, That you may have when you search a question that is not available, You get more options with the MDN API to link you to the Docs.

Search Filter
There are multiple search filters to help you narrow down your search. Search By Title allows for a more language precise search The Search By Code help's with finding languages that solves the code question. Remove Code Lable removes the language Name Disable Option Slide disables the slide option and Codesaved is a button to access user saved answers

Youtube Search
Show More Help displays more answers if available. Youtube search is available in the show more Help section. When checked it will display code answers on youtube that can help understand the question better Also having a Video help can be usefull.

Mdn Search
Mdn Search uses the Mdn API to display answers when the app doesnt have an answers also the mdn option can be checked in the sideMenu. Multiple answers are showen in divs and the link to each page is connected also the search menu is added to the start of the page for easy control to find your desired answers quickly.
